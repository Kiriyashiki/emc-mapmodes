# EMC Mapmodes

Display various statistics relating to [EarthMC](https://earthmc.net/) towns and nations directly on map with colors.

Hosted at https://emc-map.hainaut.xyz/

![Map Preview English](./public/featurepresentation-en.png)
![Map Preview Japanese](./public/featurepresentation-ja.png)

## Features
- Multiple map modes to select
- Popups with stats
- Translations in multiple languages
- Chunk Grid
